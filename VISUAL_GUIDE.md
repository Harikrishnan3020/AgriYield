## 🔄 User Flow

### Buying Medicines Flow:
```
Dashboard (Home)
    ↓
Scan Crop / View Results
    ↓
Results Card Shows:
  • Disease: "Late Blight"
  • Treatment info
  • [Buy Medicines] button ← Click here
    ↓
Medicine Dashboard Page
  • Shows 4 curated medicines
  • Each with detailed info
  • Click "View Full Details"
    ↓
Medicine Detail Modal
  • Complete specifications
  • Benefits & precautions
  • [Add to Cart] button
```

### Profile Enhancement Flow:
```
Bottom Nav → Profile Tab
    ↓
Enhanced Profile Header
  • Avatar with camera button
  • Level badge
  • Quick stats
    ↓
Three Tabs: [Overview] [History] [Badges]
    ↓
Overview: Insights + Edit + Account Info
History: Past scans with details
Badges: Achievement system
```

## 🎨 Color Coding

**Medicine Types:**
- 🔵 Blue = Fungicide
- 🔴 Red = Pesticide  
- 🟡 Yellow = Herbicide
- 🟢 Green = Organic

**Severity Levels (History):**
- 🔴 Red = High risk
- 🟠 Amber = Medium risk
- 🟢 Green = Low/Healthy

**Achievement States:**
- ⭐ Gold gradient = Unlocked
- ⚫ Grayscale = Locked

## 📊 Key Metrics

**Medicine Dashboard:**
- 4 medicine options
- 8+ data points per medicine
- Real-time stock status
- Distance-based sorting

**Profile Enhancements:**
- 3 navigation tabs
- 4 achievement badges
- 3 scan history items
- 6 profile sections
- User level calculation
