## 🎯 Common Tasks

### 1. Diagnose a Crop Disease
```
1. Home Screen
2. Click "Take Photo" or "Upload Image"
3. Capture/Select crop image
4. Wait 3-5 seconds for analysis
5. View results
```

### 2. Get AI Farming Advice
```
1. Quick Access → AI Assistant
2. Type question (e.g., "How to prevent blight?")
3. Get instant response
4. Use quick replies for follow-ups
```

### 3. Check Weather for Farming
```
1. Quick Access → Weather
2. View Today's conditions
3. Check 7-day forecast
4. Read farming advice
5. Plan activities accordingly
```

### 4. View Your Analytics
```
1. Quick Access → Analytics
2. Select time range (week/month/year)
3. Review charts and trends
4. Read AI insights
5. Export data (optional)
```

### 5. Buy Crop Medicines
```
1. After diagnosis → "Buy Medicines"
OR Quick Access → Medicines
2. Browse recommended products
3. Click "View Full Details"
4. Review information
5. Click "Buy Now"
```

## 🔧 Troubleshooting

### Camera Not Working?
1. Allow camera permissions
2. Refresh the page
3. Try "Upload Image" instead

### AI Not Responding?
1. Check internet connection
2. Verify API keys in .env
3. Refresh the page

### Build Errors?
```bash
# Clear cache and reinstall
rm -rf node_modules package-lock.json
npm install
npm run dev
```

### Video Not Loading?
- Videos show fallback with YouTube search
- Click "Search on YouTube" button
- Or click "Retry"
